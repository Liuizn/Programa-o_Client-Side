Linguagens de Programação (Surgimento, vantagens, sites que utilizam)


CSS

O Cascading Style Sheet, traduzido para o Português (Folhas de estilo em cascatas). É uma maneira de dar estilo a sua página Web ao seu código HTML. Quando o HTML foi criado, a intenção não era de forma alguma, formatar informação, de acordo com a evolução foi ficando mais complexo, Por esse tempo apareceu o salvador da pátria. Håkon Wium Lie, vendo toda essa dificuldade, resolveu criar um jeito mais fácil para formatar a informação. Foi aí que ele propôs a criação do CSS ou Cascading Style Sheets… Esse era o ano de 1994. Todas as páginas web utilizam o CSS.

PHP

Hypertext Preprocessor é uma linguagem de script open source de uso geral, muito utilizada, e especialmente adequada para o desenvolvimento web e que pode ser embutida dentro do HTML. O PHP como é conhecido hoje, é na verdade o sucessor para um produto chamado PHP/FI. Criado em 1994 por Rasmus Lerdof, a primeira encarnação do PHP foi um simples conjunto de binários Common Gateway Interface (CGI) escrito em linguagem de programação C. Wordpress, Wikipedia Flickr Goolgle Spotify Uber e etc.

HTML

O HTML é uma das linguagens que envolvem o desenvolvimento de um site ou aplicativo, diferente do que muita gente acha, HTML não é uma linguagem de programação assim como o PHP, mas sim uma linguagem de marcação. A linguagem surgiu logo que a internet começou a se popularizar, por volta dos anos 90 pois havia a necessidade de padronizar uma linguagem para marcação dos sites, o Mosaic foi o primeiro navegador a utilizar o padrão que logo depois fora adotados por todos os demais até os dias de hoje. Todas as páginas web utilizam o HTML.

TypeScript

O TypeScript é uma linguagem de código aberto que se baseia em JavaScript, uma das ferramentas mais usadas do mundo, adicionando definições de tipo estático. ipos fornecem uma maneira de descrever a forma de um objeto, fornecendo melhor documentação e permitindo que o TypeScript valide se seu código está funcionando corretamente. Escrever tipos pode ser opcional no TypeScript, porque a inferência de tipo permite que você obtenha muito poder sem escrever código adicional.Anders Hejlsberg, arquiteto da linguagem C# e criador das linguagens Delphi e Turbo Pascal, trabalhou no desenvolvimento do TypeScript. Surgido em 1 de outubro de 2012.

JavaScript

O JavaScript foi criado na década de 90 por Brendan Eich a serviço da Netscape. Essa década foi um período de revolução, pois os browsers ainda eram estáticos. O navegador mais popular dessa época era o Mosaic, da NCSA. Depois que o JavaScript foi criado, a Microsoft criou, em Agosto de 1996, uma linguagem idêntica para ser usada no Internet Explorer 3. Para conter a ambição da Microsoft, a Netscape decidiu normatizar a linguagem através da organização ECMA International, companhia que era especializada em padrões e normativas. Os trabalhos em cima da normativa ECMA-262 começaram em Novembro de 1996. O nome JavaScript já era patenteado pela Sun Microsystems (hoje Oracle) e não poderia ser usado. Portanto, o nome composto por ECMA e JavaScript foi usado, resultando em ECMAScript. Mesmo com esse nome, até hoje a linguagem é conhecida carinhosamente por JavaScript. ECMAScript só é usado para se referir às versões da linguagem. Hoje estamos na versão ECMA7. Todas as páginas da web utilizam JavaScript.
